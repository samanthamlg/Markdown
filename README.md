# ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero que permite dar formato al texto de una manera fácil y rápida sin usar programas complicados. Se usa mucho en GitHub para escribir documentación como los archivos `README.md`.

## Markdown se usa para:

- Documentar proyectos en GitHub
- Crear listas de tareas
- Hacer apuntes o tutoriales
- Escribir documentación de software
- Crear páginas web rápidas (GitHub Pages)

## ¿Cómo funciona?

Markdown utiliza símbolos simples para convertir texto plano en texto con formato.

Por ejemplo:

- Para crear títulos se usa el símbolo `#`
- Para negritas se usan `**`
- Para listas se usan `-` o números

## Ventajas de Markdown

- Es rápido de escribir
- Es fácil de leer incluso sin formato
- Se usa ampliamente en programación
- Compatible con GitHub, GitLab y muchas apps
- Perfecto para documentación técnica
